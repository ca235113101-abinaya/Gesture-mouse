Human–computer interaction has traditionally relied on devices such as keyboards and mice. However, the conventional mouse faces limitations including the need for a flat surface, reduced portability, and limited accessibility for differently-abled users. To address these challenges, this project introduces a motion-controlled mouse built using the MPU6050 sensor, which combines a 3-axis accelerometer and gyroscope, along with push-button switches for click operations.In the proposed system, the MPU6050 captures hand movements, which are processed by a microcontroller and translated into cursor motion on a computer screen through USB HID communication. The switches provide left- and right-click functions, enabling basic mouse operations without requiring a traditional hardware mouse. The prototype was tested and shown to offer reliable cursor control with minimal latency, making it both practical and costeffective.This project demonstrates that low-cost sensors and open-source hardware can be used to develop accessible and portable alternatives to conventional pointing devices. It has potential applications in assistive technology, education, gaming, and virtual reality, while also serving as a platform for future enhancements such as wireless connectivity and advanced gesture recognition.
